# Propuestas / prototipos legacy

Esta carpeta contiene maquetas HTML antiguas usadas como propuestas visuales iniciales:

- `admin.html`
- `login.html`
- `residente.html`

La versión activa del frontend ya no carga estas páginas. Las pantallas reales están en:

- `frontend/pages/login/Login.html`
- `frontend/pages/admin/Admin.html`
- `frontend/pages/residente/Residente.html`

La arquitectura actual usa módulos y fragments en:

- `frontend/js/admin/`
- `frontend/js/resident/`
- `frontend/pages/admin/sections/`
- `frontend/pages/residente/sections/`

No borrar esta carpeta sin confirmar con el equipo, porque puede servir como referencia histórica de diseño.
